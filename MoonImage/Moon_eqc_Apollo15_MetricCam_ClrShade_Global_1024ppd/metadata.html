Title: Apollo 15 Metric Cam DEM, ColorHillshade
Description:
Abstract:
The Apollo 15 DEM color-shade is generated using the colormap and hillshade functions in Vision Workbench for the DEM files. The min and max values of the colormap(legend.png)correspond to the range [-8499, 10208 in meters relative to the average moon radius of 1737.4km. The DEM colorshade values are stored as 8-bit signed integer at 14.8m/pixel resolution. These DEM colorshade files were generated using the Vision Workbench library: a software tool produced by the NASA Ames Intelligent Robotics Group(IRG)for automated processing of stereo imagery.
Purpose:
This ColorHillshade of Digital Elevation Model provides a medium resolution representation of the equatorial regions of the Moon.

Data_Quality_Information:
Attribute_Accuracy:
Attribute_Accuracy_Report:
Precision data is provided at each pixel and is available for download. A visual representation of the precision data is available in the Confidence Map.
Logical_Consistency_Report:
The Apollo 15 DEM precison maps are generated using the elevation expected data standard deviation of all the available Apollo 15 DEM files. The precision values of the A15 DEM zone are captured by the following files: a15_prec_0E_18N_1024PPD.tif ..a15_prec_176E_26S_1024PPD.tif. The DEM confidence values are stored as 8-bit signed integer at 14.8m/pixel resolution. The values in the confidence file are derived from the precision values. The value range from 10 to 14 represents poor confidence (10) to very good confidence (14), with 15 = manually edited. The confidence values in the range 10-14 represent values of the standard deviation stored in the precision files as follows: value 10 for standard deviation larger than 100m, value 11 for standard deviation in the range 75-100m, value 12 for standard deviation in the range 50-75m, value 13 for standard deviation in the range 25-50m, and value 14 for standard deviation in the range 0-25m.
Completeness_Report:
The accuracy and precision of digital elevation map is lower in areas where the images where corrupted by noise in the camera path or scanning artifacts. While the effect of image noise is reduced by the our robust stereo matching algorithm some of the artifacts remain present in the final product. Other areas of low precision and accuracy include shadow regions where stereo correlation has poor quality and areas along the image boundaries.

Process_Step:
Process_Description:
The Apollo 15 DEM color-shade is generated using the colormap and hillshade functions in Vision Workbench for the DEM files. (colormap --lut-file LMMP_color_medium.lut -o $colorShadeFile -s $shadeFile --moon --legend $demFile). The light source azimuth for the hisllshade is set to 315 degrees (0 degrees is right, and positive values increasing counter clockwise). The range of Apollo images used here include all nadir looking images of the Apollo 15 Metric zone: AS15-M-0072.cub-AS-15-M-0103.cub, AS-15-M-0136.cub-AS-15-M-0160.cub, AS15-M-0281.cub-AS15-M-1426.cub, 460-602, 872-1031, 1052-1159, 1562-2491.
